# Gemini Annotator

Gemini Annotator is a lightweight Chrome extension that uses Google Gemini AI to assist your daily reading and learning. Select any text on a webpage to start a deep conversation with AI.

## Features

- **Instant Explanation**: Select difficult terms or paragraphs to ask AI immediately.
- **Contextual Translation**: Accurate translations based on the surrounding context.
- **Lightweight & Fast**: Powered by Gemini Flash models for quick responses.
- **Native Experience**: Modern UI designed to blend perfectly into the browser environment.

## Installation

No complex configuration required. Download and use immediately:

1. **Download**: Go to the [Releases Page](这里填你的Release页面链接) of this repository and download the latest `.zip` package.
2. **Unzip**: Extract the zip file to a safe location on your computer.
3. **Load Extension**:
    - Open Chrome and go to `chrome://extensions`.
    - Enable "Developer mode" in the top right corner.
    - Click "Load unpacked".
    - Select the folder you just extracted.
4. **Start Using**: Refresh any webpage, select some text, and click the purple button that appears.
